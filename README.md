# 🧩 PennyWise – Smart Expense Tracker

**PennyWise** is a **smart, AI-powered expense tracker** that helps you manage your money efficiently. Track your daily expenses, set personalized budgets, scan receipts using OCR, get intelligent financial insights, and visualize your spending habits — all in one place.

---

## 🚀 Features

### 🔹 **Core Features**

* 📊 **Smart Expense Tracking** – Add, edit, and delete transactions easily.
* 📂 **Cloud Sync** – Real-time storage and synchronization using **Firebase**.
* 🧾 **OCR-Powered Receipt Scanner** – Upload or capture receipts to auto-extract expense details.
* 🎙️ **Voice Input Support** – Add expenses hands-free using voice commands.
* 💸 **Multi-Currency Support** – Fetches **real-time currency rates** via API.

### 🤖 **AI-Powered Insights**

* 📈 **Spending Analysis** – Get monthly, weekly, and daily breakdowns.
* ⚡ **Budget Recommendations** – Personalized budgets based on past spending.
* 🔔 **Smart Alerts** – Notifications when you’re about to exceed your budget.

### 🔐 **Security**

* 🔑 **Firebase Authentication** – Secure login with email, password, or Google OAuth.
* 🔒 **Data Encryption** – Ensures sensitive financial data stays private.

## 📦 Installation

### **1. Clone the Repository**

```bash
git clone https://github.com/your-username/PennyWise.git
cd PennyWise
```

### **2. Install Dependencies**

```bash
npm install
```

### **3. Configure Firebase**

* Go to [Firebase Console](https://console.firebase.google.com/)
* Create a new project
* Get your Firebase config and paste it into `/config/firebase.js`

### **4. Run the App**

```bash
npm start
```
